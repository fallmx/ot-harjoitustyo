# Ohjelmistotekniikka, harjoitustyö

Olen tekemässä **musiikintoistosovellusta**, jonka ominaisuudet tulevat keskittymään musiikin kanssa tahdissa *soittamiseen*.

## Käynnistys

```
poetry install
poetry run invoke start
```
